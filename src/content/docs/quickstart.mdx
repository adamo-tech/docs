Get your Jetson streaming video in under 5 minutes. This guide assumes you're SSH'd into a Jetson (Orin Nano, Orin NX, AGX Orin, Thor, etc.) with a USB camera connected. If you have a CSI camera, ROS camera topic, or another source, see the [Cameras](/cameras/) guide.

## Install

```sh
curl -sL https://install.adamohq.com | sh
```

This downloads the `adamo` binary for your platform and adds it to your PATH.

## Get an API Key

Sign up at [operate.adamohq.com](https://operate.adamohq.com) and go to **Settings > API Keys** to create a key. Copy it — you'll need it in the next step.

## Create a Config

Create `~/adamo/config.yaml`:

```yaml
api_key: "ak_your_key_here"
robot_name: "my-robot"

video_tracks:
  - name: "main"
    source_type: "v4l2"
    v4l2_device: "/dev/video0"
    encoder: "auto"
    bitrate: 4000
    fps: 30
```

`robot_name` is how the robot appears in the web interface.

## Run

```sh
adamo --config ~/adamo/config.yaml
```

A successful startup looks like:

```
Adamo Network                    v0.1.0

Robot:        my-robot
Org:          my-org
View Robot:   https://operate.adamohq.com/#/operate/my-robot
```

## See Your Robot

Open [operate.adamohq.com](https://operate.adamohq.com) and log in. Your robot will appear in the grid with its video stream already playing. Click it to start operating.
